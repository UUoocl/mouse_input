External Event Listener

This page demonstrates how external applications or separate browser windows can consume the mouse events broadcast by the main Mouse Monitor.

How to use:

  1. Open mouse_monitor.html in one window (or inside OBS).
  2. Open this page (listener.html) in a separate browser tab or window.
  3. Trigger mouse events in the monitor. You will see them appear in the logs below via the BroadcastChannel API.
